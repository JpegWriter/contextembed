# reports/

This directory contains IPTC/XMP audit reports generated by `npm run iptc:audit`.

## Files

- `iptc_audit_<timestamp>.json` - Full machine-readable audit results
- `iptc_audit_<timestamp>.csv` - Spreadsheet-friendly summary

## Report Structure

### JSON Report

```json
{
  "generatedAt": "2024-01-15T10:30:00.000Z",
  "summary": {
    "total": 100,
    "passed": 85,
    "failed": 15,
    "passRate": "85.0",
    "topMissing": [["Credit", 10], ["RightsUsageTerms", 5]],
    "topWarnings": [["Caption too short", 20]],
    "hasContextEmbed": 80,
    "avgTitleLen": 45,
    "avgCaptionLen": 350,
    "avgKeywordCount": "8.5"
  },
  "results": [
    {
      "filename": "photo_001.jpg",
      "pass_required": true,
      "missing_required": [],
      "warnings": [],
      "title_len": 42,
      "caption_len": 380,
      "keyword_count": 12,
      "has_contextembed_namespace": true,
      "has_usage_terms": true
    }
  ]
}
```

### CSV Columns

| Column | Description |
|--------|-------------|
| filename | Image filename (no path for GDPR) |
| pass_required | Boolean - all required fields present |
| missing_required | Semicolon-separated list of missing fields |
| warnings | Semicolon-separated list of warnings |
| title_len | Character count of ObjectName/Title |
| caption_len | Character count of Caption-Abstract |
| keyword_count | Number of keywords |
| has_contextembed | Boolean - has XMP-contextembed namespace |
| has_usage_terms | Boolean - has usage terms in any field |

## GDPR Compliance

Reports only contain:
- Filenames (no full paths)
- Metadata field presence/absence
- Field lengths and counts

Reports do NOT contain:
- Full file paths
- IP addresses
- User identifiers
- Actual metadata content (to avoid leaking personal data in captions)
